# Gestion de Stock

> Application de bureau Python/Tkinter pour gérer un stock de produits enregistré dans un fichier CSV.

## Lancer l'application

Depuis le dossier du projet, exécutez :

```bash
python3 main.py
```

## Identifiants de connexion

| Champ             | Valeur     |
| ----------------- | ---------- |
| Nom d'utilisateur | `admin`    |
| Mot de passe      | `admin123` |

## Fonctionnalites

- Ajouter, modifier et supprimer des produits
- Rechercher par nom, catégorie ou fournisseur
- Enregistrer les données dans `stock.csv`
